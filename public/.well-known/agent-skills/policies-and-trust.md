# Pack Policies And Trust

Use this skill when an agent needs Pack's public legal, privacy, accessibility, or trust materials.

## Primary sources

- Privacy Policy: `https://www.trypackai.com/privacy`
- Terms of Service: `https://www.trypackai.com/terms`
- Accessibility statement: `https://www.trypackai.com/accessibility`
- Security contact: `https://www.trypackai.com/.well-known/security.txt`

## Guidance

1. Prefer the dedicated policy pages over summarizing from memory.
2. Use the accessibility page for accommodation questions and known support channels.
3. Use `security.txt` only for vulnerability-reporting and security-contact workflows.
