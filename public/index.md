# Pack

Pack is an AI travel assistant for turning scattered travel inputs into a trip you can review in one place.

## What Pack does

- Organizes travel confirmations and booking details
- Builds trip drafts from prompts, confirmations, calendars, and preferences
- Helps travelers review flights, hotels, schedules, and supporting trip context together
- Supports booking-oriented travel planning across web and mobile

## Core value

One prompt to get to the flights, hotel, and schedule you would likely choose yourself, without managing the trip across too many separate tools.
